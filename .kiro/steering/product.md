# Product Overview

This is a personal portfolio and service website for "heyIsmail" - an AI-powered MVP development service targeting founders and entrepreneurs.

## Core Value Proposition
- Build MVPs in 7 days using AI + human design
- Fixed pricing ($5K-15K) with no surprises
- Complete ownership transfer (code, deployment, documentation)
- Focus on investor-ready products that generate real traction

## Target Audience
- Founders with ideas needing validation
- Entrepreneurs seeking rapid MVP development
- Startups requiring technical expertise without long-term commitments

## Key Features
- Portfolio showcase of completed projects
- Service comparison with alternatives (agencies, freelancers, no-code)
- Transparent pricing structure
- Client testimonials and success metrics
- Direct booking integration via Cal.com

## Brand Tone
- Professional yet approachable
- Results-focused and no-nonsense
- Emphasizes speed, quality, and transparency
- Uses modern, tech-savvy language
