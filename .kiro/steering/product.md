# Product Overview

This is a personal portfolio and service website for a freelance developer who specializes in building MVPs for non-technical founders. The site showcases a 7-day MVP development service with a focus on complete ownership transfer and modern tech stack implementation.

## Key Value Proposition
- 7-day MVP development sprints
- Complete code ownership transfer
- Modern tech stack (Next.js, React, TypeScript)
- Focus on non-technical founders
- Concept builds demonstrating skills

## Target Audience
Non-technical founders with validated ideas who need rapid MVP development without the typical agency overhead or learning curves.
